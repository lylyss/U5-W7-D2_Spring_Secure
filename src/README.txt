PROGETTO VIAGGI AZIENDALI

applicazione Spring Boot per gestione di viaggi di lavoro


ho utilizzato:

-IntelliJ
-Maven
-PostgreSQL
-Cloudinary 
-Postman
-drawsql

application.properties carica i dati da env.properties

api disponibili per Viaggi, Dipendeti e Prenotazioni

GET POST PUT DELETE PATCH 

validazioni @NotNull, @NotBlank e @Email

exception:
-BadRequestException
-GlobalException
-ResourceNotFoundException
-ValidationException
